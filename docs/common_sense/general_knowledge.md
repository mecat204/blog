### 000. commom sense
* **common sense**

```
常识 common sense
 └─ 常识性知识 general knowledge
     └─ 国际知识 international knowledge
         └─ 国家代码 ISO country codes
```

* **归纳**
| 类别名称       | 英文术语                                                    | 说明                            |
| ---------- | ------------------------------------------------------- | ----------------------------- |
| **国际标准代码** | **ISO country codes**                                   | 国际标准化组织定义的国家/地区代码（如 ISO 3166） |
| **国际通用知识** | **general international knowledge**                     | 跨文化或全球通用的信息，如国名缩写、国旗、首都等      |
| **常用缩写**   | **abbreviations** 或 **country abbreviations**           | 国家常用的英文缩写（如 USA, UK）          |
| **地理常识**   | **geographical knowledge** 或 **geopolitical awareness** | 关于国家、地区、地理位置的基础常识             |
| **编码系统知识** | **code systems / standard codes**                       | 属于编码系统知识范畴，适用于程序设计、物流、数据交换等   |

* **分类**
| 场景         | 建议翻译              |
| ---------- | ----------------- |
| 指人的判断、理性   | common sense      |
| 指文化历史类基础知识 | general knowledge |
| 指某一领域初步了解  | basic knowledge   |

### 003 └─ 是一种 Unicode 表示的"树状结构符号"
* **具体字符说明**
| 符号  | 名称                              | Unicode 编码 | 常见用途           |
| --- | ------------------------------- | ---------- | -------------- |
| `└` | Box Drawings Light Up and Right | U+2514     | 表示“树状结构”的末端或分支 |
| `─` | Box Drawings Light Horizontal   | U+2500     | 表示横线连接         |

> 组合在一起 └─ 表示“某个层级下的子项”，在树状结构中使用频率很高。

* **输入方法（几种方式）**
  - 复制

    ```
    ├─
    └─
    │
    ```

  - 使用 Mac / Windows 特殊输入方式
    * Windows
      - 打开“字符映射表”（按 Win + R，输入 charmap，回车），选择字体如 Courier New 或 Lucida Console，找到这些符号。
      - 或使用 Alt 编码：
        * └: 按住 Alt，小键盘输入 192（确保启用小键盘）
        *─: Alt + 196

    * Mac:
      * 使用"表情与符号"（⌃ Control + ⌘ Command + Space），搜索 box drawing。

  - 在编程中使用 Unicode 编码
    * 如果你在写代码或 Markdown，想用程序插入：

    ``` python
    print("\u2514\u2500")  # 输出 └─
    ```

### 005. 关键词的重复可植入潜意识形成信念
* **但丁的作品充满了大量的重复，他为什么这么写**
  - 口头传统与韵律结构
    * 但丁的作品创作于中世纪，当时口头传播仍很重要。他采用了“三行诗节”（terza rima）格式，这种押韵模式（aba bcb cdc...）天然带有某种旋律性和重复性，使听众更容易理解与记忆。

  - 强调与呼应（强调观念）
    * 重复常用于：
      - 强调重要概念，例如天堂中的“爱”、地狱中的“惩罚”、炼狱中的“悔过”。
      - 结构对称与呼应，如但丁三部曲每一部分都由33歌（外加一首引子），层层递进、互为镜像，形成一种神圣对称感。

  - 神学与象征意义
    * 中世纪基督教文化中，重复具有神圣象征意义，如：
      - “三”代表圣父、圣子、圣灵。
      - 多次出现相似的情节或词语，强化了**“永恒性”“普遍性”**的宗教氛围。
      > 例如，《神曲》中经常重复“正义”、“恩典”、“秩序”等词，是对神的属性的不断“颂扬”与呼应。

  - 心理与修辞效应
    * 但丁通过重复，营造了一种诗意与心理共鸣：
      - 引导读者逐渐沉浸于地狱的恐惧、炼狱的挣扎、天堂的喜悦。
      - 同样场景中重复出现某些语言，强化了情绪效果与主题张力。

  - 艺术中的“循环”结构
    * 但丁深受古典哲学影响（如柏拉图、亚里士多德），认为宇宙是有秩序和轮回结构的。他的文学结构与内容相互呼应，比如：
      - 地狱的惩罚是罪行的“镜像”；
      - 炼狱的悔罪是地狱的反面；
      - 天堂则是灵魂最终的和谐归宿。

  > 这种“循环”就通过不断重复与变形表现出来。

  - 总结:
    > 但丁之所以大量使用重复，是一种诗歌技法、修辞手段、哲学表达，也体现了他试图用文学表达宇宙秩序与神圣理念的努力。他的重复不是冗余，而是一种“神圣对称”的体现。

* **重复最深层的心理与修辞功能之一：通过重复，强化认知、影响潜意识、塑造信念。**
  * **认知心理学的解释：熟悉性偏好（Mere Exposure Effect）**
    > 人类对反复接触的词语或观念，更容易产生接受、理解、甚至喜爱的情绪。
    * 在这种效应下，某个词或概念被反复提及，就容易被大脑归入“熟悉、可信”的范畴。例如：
      - 反复看到“光明”、“拯救”这类词，会强化对天堂的向往。
      - 在地狱篇中重复“惩罚”、“罪行”，会让人对罪恶的恐惧根植于心。

  * **潜意识认同的形成：情绪+语言重复**
    > 重复不仅传递信息，更是潜移默化地构建价值观的一种方式。
    * 但丁通过不断重复某些词或句式，例如：
      - "正义之神驱我而来"（刻在地狱门上）
      - "爱促成这一切"（天堂篇中频繁提及）

  * 反复出现之后，这些词不仅具有宗教意义，还成为读者的内在语言（inner speech），潜移默化地塑造其对世界的理解。

  * **古典修辞术中的“反复法”（Anaphora / Repetition）**
    > 在古典修辞中，重复被视为一种激发情感、建立节奏、强化观点的有力武器。

    * 比如但丁每一节结尾押韵的词语，会在下一节再次出现，造成一种连续的回响。这就像一首永不停歇的赞美诗，最终让你不得不“记住它、感受它、相信它”。

* **现实类比：**
  - 我们可以看看现代的类比：
    * 广告语不断重复品牌名称，就是为了让你“潜意识地记住它”。
    * 演讲中重复“我有一个梦想”（马丁·路德·金）是一种情绪催化。
    * 宗教仪式中不断重复祷词，是为了建立信仰上的认同与共鸣。

* **回到但丁**
  - 因此，在《神曲》中，重复并非只是文学手法，而是一种**“灵魂的再教育”**。通过不断重复
    * 强化道德判断（何为罪、何为德）
    * 植入神圣价值（爱、秩序、恩典）
    * 引导灵魂认同“上升之路”

* **结语**
  - **关键词的重复可植入潜意识形成信念**
  - **但丁之所以伟大的原因之一——他不仅写出一部诗歌，更构建了一个认知系统，读者在阅读中，已被悄悄地“重塑”。**

### 006. 重复本身
> "不断重复"确实是宗教传播中一个常见而强效的工具
* **既可以是"洗脑"的手段，也可以是"信仰建立"或"价值教育"的方式，这取决于目的、语境和你怎么看待它。**

* **宗教为什么反复重复？**
  - **重复是记忆之母**
    * 宗教典籍（如《圣经》《古兰经》《佛经》）大量使用重复句式、叙事轮回、重复训诫，目的是让信徒记住关键教义。
    * 例如《圣经》中常见的“不要惧怕”、“主是慈爱的”一再重复，就是强化信仰与情绪支撑。

  - **潜意识植入 + 情感塑造**
    * 仪式性语言：祈祷、诵经、唱赞美诗等形式，本质上都是“重复+节奏”的结合，具有“情感催眠”作用。
    * 反复听、反复说，久而久之，信徒就会将这些理念内化为自我认知的一部分。

* **是"洗脑"吗？**
  - 如果站在批判角度（尤其是现代启蒙思想视角）：
    * 宗教确实利用了“重复 + 权威”来影响个体，使其对某些观念不加质疑。
    * 这种方式在某些极端宗教群体中，甚至被用作操控手段，这时“洗脑”一词就很贴切了。

  - 如果站在信徒角度：
    * 重复是一种“修心”、“调频”，帮助人在复杂、苦难中获得内在秩序感。
    * 祷告不是洗脑，而是连接神圣、净化心灵。

* **再看现代例子**
  - **"重复影响认知"并非宗教专属:**
    * 广告每天"洗脑式"播放品牌词句;
    * 政治口号反复喊："人民万岁"、"国家安全";
    * 自我暗示（affirmation）练习也用重复: 比如"我可以"、"我值得"。

* **总结**
| 角度     | 对重复的看法             |
| ------ | ------------------ |
| 宗教批判视角 | 重复 = 洗脑 = 思想控制     |
| 宗教学视角  | 重复 = 修行 = 建立信仰认同   |
| 认知心理学  | 重复 = 加强记忆 = 建构自我观念 |
| 文学修辞角度 | 重复 = 节奏美感 = 情绪强化手段 |

> 所以说，重复本身不是“好”或“坏”，关键在于它的内容、动机与接受者的主动性。
