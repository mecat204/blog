### 0001. 通用（General）
* **基础操作**
  - 新建文件：Ctrl + N
  - 打开文件：Ctrl + O
  - 保存文件：Ctrl + S
  - 全部保存：Ctrl + K S
  - 关闭当前文件：Ctrl + W
  - 退出 VS Code：Ctrl + Q
  - [官方文档](https://code.visualstudio.com/docs/getstarted/keybindings)

---
### 0002. 编辑（Basic Editing）
* **文本编辑**
  - 剪切当前行：Ctrl + X
  - 复制当前行：Ctrl + C
  - 向下复制一行：Shift + Alt + ↓
  - 向上复制一行：Shift + Alt + ↑
  - 删除当前行：Ctrl + Shift + K
  - 向下移动一行：Alt + ↓
  - 向上移动一行：Alt + ↑
  - 撤销：Ctrl + Z
  - 重做：Ctrl + Y

---
### 0003. 多光标与多行编辑（Multi-cursor）
* **多光标操作**
  - 添加光标（鼠标）：Alt + Click
  - 向下添加光标：Ctrl + Alt + ↓
  - 向上添加光标：Ctrl + Alt + ↑
  - 选择下一个相同单词：Ctrl + D
  - 选择所有相同单词：Ctrl + Shift + L
  - 撤销上一个光标：Ctrl + U
  - [多光标说明](https://code.visualstudio.com/docs/editor/codebasics#_multiple-selections-multicursor)

---
### 0004. 选择（Selection）
* **选区控制**
  - 选中整行：Ctrl + L
  - 扩大选区：Shift + Alt + →
  - 缩小选区：Shift + Alt + ←
  - 匹配括号选区：Ctrl + Shift + \
  - 全选：Ctrl + A

---
### 0005. 搜索与替换（Search）
* **查找功能**
  - 当前文件查找：Ctrl + F
  - 当前文件替换：Ctrl + H
  - 全局搜索：Ctrl + Shift + F
  - 全局替换：Ctrl + Shift + H
  - 下一个结果：F3
  - 上一个结果：Shift + F3

---
### 0006. 文件与窗口（File / Window）
* **窗口管理**
  - 快速打开文件：Ctrl + P
  - 打开最近文件：Ctrl + R
  - 切换编辑器组：Ctrl + 1 / 2 / 3
  - 拆分编辑器：Ctrl + \
  - 关闭全部编辑器：Ctrl + K W

---
### 0007. 代码导航（Navigation）
* **跳转相关**
  - 跳转到定义：F12
  - 跳转到声明：Ctrl + F12
  - 预览定义：Alt + F12
  - 返回上一位置：Alt + ←
  - 前进：Alt + →
  - 查看引用：Shift + F12

---
### 0008. 智能提示与重构（IntelliSense）
* **代码智能**
  - 触发提示：Ctrl + Space
  - 快速修复：Ctrl + .
  - 重命名符号：F2
  - 格式化文档：Shift + Alt + F
  - 格式化选中代码：Ctrl + K Ctrl + F

---
### 0009. 行操作（Line）
* **行控制**
  - 缩进：Ctrl + ]
  - 反缩进：Ctrl + [
  - 跳转到指定行：Ctrl + G
  - 在下方插入空行：Ctrl + Enter
  - 在上方插入空行：Ctrl + Shift + Enter

---
### 0010. 注释（Comments）
* **注释操作**
  - 行注释：Ctrl + /
  - 块注释：Shift + Alt + A
  - 下一个注释：Ctrl + Shift + ]
  - 上一个注释：Ctrl + Shift + [

---
### 0011. 终端（Terminal）
* **集成终端**
  - 打开终端：Ctrl + `
  - 新建终端：Ctrl + Shift + `
  - 切换终端：Ctrl + PageUp / PageDown
  - 关闭终端：Ctrl + Shift + W

---
### 0012. 面板与视图（Views）
* **界面切换**
  - 命令面板：Ctrl + Shift + P
  - 资源管理器：Ctrl + Shift + E
  - 搜索面板：Ctrl + Shift + F
  - Git 面板：Ctrl + Shift + G
  - 调试面板：Ctrl + Shift + D
  - 扩展面板：Ctrl + Shift + X

---
### 0013. 调试（Debug）
* **调试控制**
  - 启动调试：F5
  - 停止调试：Shift + F5
  - 单步跳过：F10
  - 单步进入：F11
  - 继续执行：F5

---
### 0014. 折叠（Folding）
* **代码折叠**
  - 折叠当前区域：Ctrl + Shift + [
  - 展开当前区域：Ctrl + Shift + ]
  - 折叠全部：Ctrl + K Ctrl + 0
  - 展开全部：Ctrl + K Ctrl + J
