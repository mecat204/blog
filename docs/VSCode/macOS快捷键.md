### 0001. 通用（General）
* **基础操作**
  - 新建文件：⌘ + N
  - 打开文件：⌘ + O
  - 保存文件：⌘ + S
  - 全部保存：⌘ + Option + S
  - 关闭当前文件：⌘ + W
  - 退出 VS Code：⌘ + Q
  - [官方文档](https://code.visualstudio.com/docs/getstarted/keybindings)

---
### 0002. 编辑（Basic Editing）
* **文本编辑**
  - 剪切当前行：⌘ + X
  - 复制当前行：⌘ + C
  - 向下复制一行：Shift + Option + ↓
  - 向上复制一行：Shift + Option + ↑
  - 删除当前行：⌘ + Shift + K
  - 向下移动一行：Option + ↓
  - 向上移动一行：Option + ↑
  - 撤销：⌘ + Z
  - 重做：⌘ + Shift + Z

---
### 0003. 多光标与多行编辑（Multi-cursor）
* **多光标操作**
  - 添加光标（鼠标）：Option + Click
  - 向下添加光标：⌘ + Option + ↓
  - 向上添加光标：⌘ + Option + ↑
  - 选择下一个相同单词：⌘ + D
  - 选择所有相同单词：⌘ + Shift + L
  - 撤销上一个光标：⌘ + U
  - [多光标说明](https://code.visualstudio.com/docs/editor/codebasics#_multiple-selections-multicursor)

---
### 0004. 选择（Selection）
* **选区控制**
  - 选中整行：⌘ + L
  - 扩大选区：Shift + Option + →
  - 缩小选区：Shift + Option + ←
  - 匹配括号选区：⌘ + Shift + \
  - 全选：⌘ + A

---
### 0005. 搜索与替换（Search）
* **查找功能**
  - 当前文件查找：⌘ + F
  - 当前文件替换：⌘ + Option + F
  - 全局搜索：⌘ + Shift + F
  - 全局替换：⌘ + Shift + H
  - 下一个结果：⌘ + G
  - 上一个结果：⌘ + Shift + G

---
### 0006. 文件与窗口（File / Window）
* **窗口管理**
  - 快速打开文件：⌘ + P
  - 打开最近文件：⌘ + R
  - 切换编辑器组：⌘ + 1 / 2 / 3
  - 拆分编辑器：⌘ + \
  - 关闭全部编辑器：⌘ + K W

---
### 0007. 代码导航（Navigation）
* **跳转相关**
  - 跳转到定义：F12
  - 跳转到声明：⌘ + F12
  - 预览定义：Option + F12
  - 返回上一位置：Control + -
  - 前进：Control + Shift + -
  - 查看引用：Shift + F12

---
### 0008. 智能提示与重构（IntelliSense）
* **代码智能**
  - 触发提示：⌘ + Space
  - 快速修复：⌘ + .
  - 重命名符号：F2
  - 格式化文档：Shift + Option + F
  - 格式化选中代码：⌘ + K ⌘ + F

---
### 0009. 行操作（Line）
* **行控制**
  - 缩进：⌘ + ]
  - 反缩进：⌘ + [
  - 跳转到指定行：⌘ + G
  - 在下方插入空行：⌘ + Enter
  - 在上方插入空行：⌘ + Shift + Enter

---
### 0010. 注释（Comments）
* **注释操作**
  - 行注释：⌘ + /
  - 块注释：Shift + Option + A
  - 下一个注释：⌘ + Shift + ]
  - 上一个注释：⌘ + Shift + [

---
### 0011. 终端（Terminal）
* **集成终端**
  - 打开终端：⌃ + `
  - 新建终端：⌘ + Shift + `
  - 切换终端：⌘ + PageUp / PageDown
  - 关闭终端：⌘ + Shift + W

---
### 0012. 面板与视图（Views）
* **界面切换**
  - 命令面板：⌘ + Shift + P
  - 资源管理器：⌘ + Shift + E
  - 搜索面板：⌘ + Shift + F
  - Git 面板：⌘ + Shift + G
  - 调试面板：⌘ + Shift + D
  - 扩展面板：⌘ + Shift + X

---
### 0013. 调试（Debug）
* **调试控制**
  - 启动调试：F5
  - 停止调试：Shift + F5
  - 单步跳过：F10
  - 单步进入：F11
  - 继续执行：F5

---
### 0014. 折叠（Folding）
* **代码折叠**
  - 折叠当前区域：⌘ + Shift + [
  - 展开当前区域：⌘ + Shift + ]
  - 折叠全部：⌘ + K ⌘ + 0
  - 展开全部：⌘ + K ⌘ + J
