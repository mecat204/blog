* **输出佛像**
  - 可以将这段代码保存成 .bat（Windows）或 .sh（Linux/macOS）文件，运行即可。


* **佛祖保佑脚本（buddha.sh）带颜色版**
  - **使用方法（适用于 Git Bash、WSL、Linux、macOS）**
    * 保存为文件：命名为 buddha.sh
    * 赋予执行权限：

    ``` bash
    chmod +x buddha.sh
    ```

    * 运行：

    ``` bash
    ./buddha.sh
    ```

``` bash
#!/bin/bash

# 定义颜色
RED='\033[0;31m'
YELLOW='\033[1;33m'
GREEN='\033[0;32m'
NC='\033[0m' # No Color

# 输出佛像图案
echo -e "${YELLOW}"
cat << "EOF"
        _ooOoo_
       o8888888o
       88" . "88
       (| -_- |)
       O\  =  /O
    ____/`---'\____
  .'  \\|     |//  `.
 /  \\|||  :  |||//  \
/  _||||| -:- |||||-  \
|   | \\\  -  /// |   |
| \_|  ''\---/''  |_/ |
\  .-\__  `-`  ___/-. /
___`. .'  /--.--\  `. . __
."" '<  `.___\_<|>_/___.' >'""
| | :  `- \`.;`\ _ /`;.`/ - ` : | |
\  \ `-.   \_ __\ /__ _/   .-` /  /
=====`-.____`-.___\_____/___.-`____.-'======
                   `=---='
EOF

# 输出祝福语
echo -e "${GREEN}"
echo "^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^"
echo "         佛祖保佑       永无BUG"
echo "^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^"

# 重置颜色
echo -e "${NC}"
read -p "按回车退出..."

```
