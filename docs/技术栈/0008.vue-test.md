### 编程中的含义（Programming）
**[004. 编程中的含义（Programming）]{g}**

createApp()

用于：

> 创建一个 Vue Application 实例。

它并不会立即把页面显示出来，而是先创建整个应用对象。

返回值：

```javascript
const app = createApp(App)
```

其中：

```
app
```

是：

> Application Instance（应用实例）

之后才能继续：

* 注册组件
* 注册插件
* 注册全局属性
* 挂载到 DOM

---
**[005. Vue 中的作用（Vue Usage）]{g}**
Vue3 每一个项目，都从：

```javascript
createApp(App)
```

开始。

流程：

```
createApp()
        │
        ▼
Application
        │
        ▼
app.mount()
        │
        ▼
DOM
```

因此：

createApp()

负责：

> 创建整个 Vue 世界。

---
**[006. API 示例（Examples）]{g}**

```javascript
import { createApp } from "vue"

import App from "./App.vue"

const app = createApp(App)

app.mount("#app")
```

链式写法：

```javascript
createApp(App).mount("#app")
```
