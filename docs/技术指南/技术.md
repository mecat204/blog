### 02. Command
1. 保存目录到文件
``` cmd
tree /f
```

1. sublime txt 删除空白行
```
^\s*$
```

1. 列出详细地列出内容（包括隐藏文件等）可以加参数：

```cmd
dir /a /s
```

- `/a`：显示所有文件，包括隐藏文件；
- `/s`：递归列出所有子目录的文件。
  如需导出目录列表到一个文本文件，可以加上重定向：

```cmd
dir /a /s > filelist.txt
```
1. 合并ts文件
``` cmd
copy /b  F:\f\*.ts  E:\f\new.ts
copy /b *.ts new.ts
```
